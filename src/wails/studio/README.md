# 🎨 Wails Studio: Interface Nativa Go-Centric

O **Wails Studio** é a interface visual premium que se expande a partir da bandeija do sistema.

## 🚀 Filosofia: Low Overhead, High Aesthetics

Ao contrário de dashboards baseados em Next.js (Node.js/V8), o Studio é construído focando em leveza e integração profunda com o Go.

## 🛠️ Stack de UI

- **Backend:** Go (Maestro).
- **Interface:** [Go-app (Wasm)](https://go-app.dev/) ou **Svelte** (Compilação estática zero-JS runtime).
- **Estética:** Glassmorphism via CSS `backdrop-filter` orquestrado pelo Wails.

## 🗂️ Funcionalidades

1. **Model Manager:** Visualização de progresso de download e instalação de GGUFs.
2. **Data Library:** Interface de seleção de Packs (Godot, Unreal, Addons).
3. **Log Stream:** Visualização em tempo real do que a IA está processando.
4. **Settings:** Configuração da Gemini API e limites de RAM.

---

_O Studio é embutido no binário principal e ativado sob demanda._
