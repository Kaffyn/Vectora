INTEGRATION ARCHITECTURE
================================================================================

Data Flow - User Query to Response:

  1. User enters query in React UI (Agent 2)
  2. IPC bridge calls backend (ipc-bridge.ts)
  3. IPC Server receives message (server.go - Agent 1)
  4. Router dispatches to handler (router.go - Agent 1)
  5. Message stored in KVStore (store.go - Agent 1)
  6. MessageService retrieves context (service.go - Agent 3)
  7. LLM Provider processes query (gemini/qwen/llama - Agent 3)
  8. Tools Engine executes decision (engine.go - Agent 3)
  9. Results stored back in KVStore (Agent 1)
  10. Response sent via IPC (Agent 1)
  11. React component updates UI (Agent 2)
  12. User sees response
