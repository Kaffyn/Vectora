INTEGRACAO ENTRE AGENTS
================================================================================

Fluxo de Dados (Query -> Response):

1. Usuario digita pergunta na UI (Agent 2 - React)
     |
2. IPC Bridge envia para backend (ipc-bridge.ts)
     |
3. IPC Server recebe mensagem (server.go - Agent 1)
     |
4. Router distribui para handler (router.go - Agent 1)
     |
5. Mensagem armazenada em KVStore (store.go - Agent 1)
     |
6. MessageService recupera contexto (service.go - Agent 3)
     |
7. LLM Provider processa query (Gemini/Qwen/Llama - Agent 3)
     |
8. Tools Engine executa decisao (engine.go - Agent 3)
     |- Filesystem/Search/Web/System/Memory
     |
9. Resultados salvos em KVStore (Agent 1)
     |
10. Resposta enviada via IPC (Agent 1)
     |
11. React component atualiza UI (Agent 2)
     |
12. Usuario ve resposta
